◆ carregarLlocs()

static List< Lloc > core.LectorJSON.carregarLlocs ( String pathFitxer)
static

Carrega una llista de llocs a partir d’un fitxer JSON amb estructura parcialment lliure.

El JSON ha de contenir un array sota la clau "llocs", on cada element pot ser un objecte de tipus Lloc o Parquing. La detecció es fa mitjançant el camp "TIPUS": "L" per Lloc i "P" per Parquing.

En el cas dels parquings, també es processen els camps opcionals "MAX_VEHICLES", "N_CARREGADORS" i "N_CARREGADORS_PRIVATS", utilitzats per generar punts de càrrega públics i privats amb tipus alternatius (ràpida/lenta).

Parámetros
pathFitxerRuta del fitxer JSON que conté les dades dels llocs.
Devuelve
Una llista de Llocs i Parquings carregats des del fitxer.

Carrega les peticions des d’un fitxer JSON amb format específic.

Cada petició ha de contenir els camps: ID, ORIGEN, DESTI, HORA_MIN_RECOLLIDA, HORA_MAX_ARRIBADA, NUM_PASSATGERS i VEHICLE_COMPARTIT.

Els camps d'ORIGEN i DESTI són IDs que es resolen a objectes Lloc mitjançant el mapa proporcionat.

Parámetros
pathFitxerRuta del fitxer JSON que conté les peticions.
llocsPerIdMapa d'objectes Lloc indexats pel seu ID.
Devuelve
Una llista de Peticions carregades correctament.
