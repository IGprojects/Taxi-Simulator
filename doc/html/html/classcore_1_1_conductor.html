◆ teBateria()

abstract boolean core.Conductor.teBateria ( double distancia,
Simulador simulador,
Mapa mapa,
LocalTime horaInici,
LocalTime horaActual )
abstract

Comprova si el vehicle té bateria suficient per fer la distància indicada. Si no en té, intenta planificar una ruta cap al pàrquing més proper per carregar.

Parámetros
distanciaLa distància que s’ha de recórrer per completar la petició.
simuladorEl simulador on s'afegiran esdeveniments si cal carregar.
mapaEl mapa que permet calcular rutes i distàncies.
horaIniciL’hora d’inici de la simulació (referència per calcular horaris).
horaActualL’hora actual del simulador, per planificar nous esdeveniments.
Devuelve
true si el vehicle té bateria suficient per fer el trajecte, false si no i ha de carregar.
Precondición
El vehicle ha d’estar inicialitzat i ubicat a un lloc vàlid dins el mapa.
Postcondición
Si no té bateria, pot iniciar una ruta cap a un pàrquing privat per carregar. Si no hi ha pàrquing, es mostra un missatge.
